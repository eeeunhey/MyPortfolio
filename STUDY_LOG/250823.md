# 🌙 React UI 학습 기록

## 1. Navbar 오류 (`useEffect is not defined`)
- **문제**: `useEffect` 사용했는데 `import` 누락 → `ReferenceError`
- **해결**:
  ```jsx
  import React, { useEffect, useState } from "react";
  ```
- **추가 수정**:
  - `window.screenY` ❌ → `window.scrollY` ✅
  - `justity-between` ❌ → `justify-between` ✅
  - `realtive` ❌ → `relative` ✅
  - `navItems.map`는 `return` 필요

## 2. Dark Mode 토글
- **문제**: 토글이 한 번만 동작하거나 `remove` 에러 발생
- **이유**:
  - `document.documentElement.remove("dark")` → 잘못된 코드  
  - 올바른 방법: `classList.remove("dark")`
  - 중복된 상태 업데이트 로직 존재
- **정리된 코드**:
  ```jsx
  const toggleTheme = () => {
    const next = !isDarkMode;
    document.documentElement.classList.toggle("dark", next);
    localStorage.setItem("theme", next ? "dark" : "light");
    setIsDarkMode(next);
  };
  ```
- **중요 개념**:  
  - `classList.add/remove/toggle()` : HTML 요소 클래스 제어
  - Tailwind `darkMode: 'class'` 설정 필수

## 3. 별 & 유성 배경 (`StarBackground`)
### 별 생성 (`generateStars`)
- **아이디어**: 화면 크기에 따라 별 개수 결정
- **속성**:
  - `size`: 별 크기 (1~3px)
  - `x, y`: 랜덤 좌표
  - `opacity`: 투명도 (0.5~1)
  - `animationDuration`: 깜빡임 속도

### 유성 생성 (`generateMeteors`)
- **코드 설명**:
  ```js
  const numberOfMeteors = 4;                 // 유성 개수
  size: Math.random() * 2 + 1,               // 크기 (1~3)
  x: Math.random() * window.innerWidth,      // 시작 X
  y: Math.random() * (window.innerHeight*0.2)// 시작 Y (상단 20%)
  delay: Math.random() * 1.5,                // 시작 지연 (s)
  animationDuration: Math.random() * 3 + 3,  // 지속시간 (s)
  ```
- **화면 출력**:
  ```jsx
  {meteors.map(meteor => (
    <div
      key={meteor.id}
      style={{
        width: meteor.size * 50 + "px",
        height: meteor.size * 2 + "px", // (두꺼움 문제 발생)
        left: meteor.x + "px",
        top: meteor.y + "px",
        animationDelay: meteor.delay + "s", // 단위 필요
        animationDuration: meteor.animationDuration + "s",
      }}
    />
  ))}
  ```

### 두꺼운 유성 해결
- 기존: `height: meteor.size * 2 + "px"` → 최대 6px
- 개선:
  ```js
  width: meteor.size * 80 + "px", // 길게
  height: "1px",                  // 얇게
  background: "linear-gradient(90deg, rgba(255,255,255,0.9), rgba(255,255,255,0))",
  borderRadius: "9999px",
  ```
- 꼬리 효과까지 추가하면 더 자연스러움 ✨

# 📌 배운 점 요약
- React 훅(`useEffect`, `useState`)은 반드시 import 필요
- `screenY` 대신 `scrollY` 사용
- `classList` API (`add`, `remove`, `toggle`)로 다크모드 제어
- 상태 관리(`useState`)와 DOM 반영을 일관되게 해야 토글이 정상 동작
- 랜덤 좌표·크기로 별/유성 효과 구현 가능
- CSS 애니메이션 키프레임이 없으면 JS에서 값 줘도 움직이지 않음
- 유성은 얇게 만들고 `linear-gradient`로 꼬리를 표현하면 자연스러움
