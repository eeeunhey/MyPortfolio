# 📘 AboutSection 코드 공부 정리

## 📌 구성
- `AboutSection` : "About Me" 섹션
- 좌측 → 자기소개 + 버튼 2개
- 우측 → 카드 3개 (Web Dev / UI/UX / Project Management)

---

## 📑 주요 태그 구조
- `<section>` : 전체 섹션 (id="about")
- `<h2>` : 큰 제목 "About ME"
- `<div grid>` : 2열 레이아웃 (왼쪽 텍스트 / 오른쪽 카드)
- `<p>` : 자기소개 문단
- `<a>` : 버튼 (Get In Touch / Download CV)
- 카드 3개 : 아이콘 + 제목 + 설명

---

## 🎨 주요 Tailwind 클래스

### 레이아웃
- `py-24 px-4` → 섹션 패딩
- `container mx-auto max-w-5xl` → 가운데 정렬, 최대 너비
- `grid grid-cols-1 md:grid-cols-2` → 반응형 1~2열
- `gap-12` → 열 사이 간격

### 텍스트
- `text-3xl md:text-4xl font-bold text-center` → 제목
- `text-primary` → 대표 색상 강조
- `text-muted-foreground` → 흐린 색상(본문)
- `font-semibold` → 약간 굵게

### 버튼
- `cosmic-button` → 커스텀 Contact 버튼
- `px-6 py-2 rounded-full border border-primary text-primary hover:bg-primary/10 transition-colors duration-300` → CV 버튼 스타일

### 카드
- `gradient-border p-6 card-hover` → 카드 스타일
- `flex items-start gap-4` → 아이콘+텍스트 배치
- `p-3 rounded-full bg-primary/10` → 아이콘 배경 원
- `h-6 w-6 text-primary` → 아이콘 크기/색
- `font-semibold text-lg` → 카드 제목
