# React Navigation Bar 구현 가이드

## 1. isScrolled 상태 추가

### 기능 설명
- 스크롤 위치에 따라 네비게이션 바의 스타일을 동적으로 변경
- 페이지 상단에서는 투명하고, 스크롤 시 배경색과 그림자 추가

### 코드 구현
```jsx
const [isScrolled, setIsScrolled] = useState(false);

useEffect(() => {
  const handleScroll = () => {
    setIsScrolled(window.scrollY > 50);
  };

  window.addEventListener('scroll', handleScroll);
  return () => window.removeEventListener('scroll', handleScroll);
}, []);
```

### 주요 포인트
- `useState`로 스크롤 상태 관리
- `useEffect`와 `addEventListener`로 스크롤 이벤트 감지
- 50px 이상 스크롤 시 상태 변경
- cleanup function으로 메모리 누수 방지

## 2. Container와 Logo 영역

### 기능 설명
- 반응형 컨테이너로 네비게이션 요소들을 감싸기
- 브랜드 로고와 포트폴리오 타이틀 표시
- Flexbox로 좌우 정렬 구현

### 코드 구현
```jsx
<nav className={cn("fixed w-full z-40 transition-all duration-300", 
  isScrolled
)}>
  <div className="container flex items-center justify-between">
    <a 
      className="text-xl font-bold text-primary flex items-center"
      href="#hero"
    >
      <span className="relative z-10">
        <span className="text-glow text-foreground">EUNHEY</span>{" "} 
        Portfolio
      </span>
    </a>
  </div>
</nav>
```

### 주요 포인트
- `fixed` 포지션으로 상단 고정
- `z-40`으로 다른 요소들 위에 배치
- `transition-all duration-300`으로 부드러운 애니메이션
- `container` 클래스로 반응형 너비 제어
- `justify-between`으로 로고와 메뉴를 양쪽 끝에 배치

## 3. Desktop Navigation 메뉴

### 기능 설명
- 데스크톱 화면에서만 표시되는 수평 메뉴
- 각 메뉴 항목은 페이지 섹션으로 스크롤 이동
- 호버 효과와 부드러운 전환 애니메이션

### 코드 구현
```jsx
{/* desktop nav */}
<div className="hidden md:flex items-center space-x-8">
  {navItems.map((item, index) => (
    <a
      key={index}
      href={item.href}
      className="text-foreground/80 hover:text-foreground transition-colors duration-200"
    >
      {item.name}
    </a>
  ))}
</div>
```

### navItems 배열
```jsx
const navItems = [
  { name: 'Home', href: '#hero' },
  { name: 'About', href: '#about' },
  { name: 'Skills', href: '#skills' },
  { name: 'Projects', href: '#projects' },
  { name: 'Contact', href: '#contact' }
];
```

### 주요 포인트
- `hidden md:flex`로 모바일에서는 숨김, 중간 크기 이상에서만 표시
- `map()` 함수로 동적 메뉴 생성
- `space-x-8`로 메뉴 항목 간 간격 설정
- 투명도(`/80`)와 호버 효과로 인터랙티브한 UX 제공
- `transition-colors`로 부드러운 색상 변화

## 4. Mobile Navigation (예정)

### 기능 설명
- 모바일 화면에서 햄버거 메뉴로 표시
- 토글 방식으로 메뉴 열기/닫기
- 전체 화면 오버레이 메뉴

### 구현 예정 기능
- 햄버거 아이콘 (`Menu` 컴포넌트)
- 닫기 아이콘 (`X` 컴포넌트)
- 모바일 메뉴 상태 관리
- 애니메이션 효과

---

## 사용된 기술 스택

### React Hooks
- `useState`: 컴포넌트 상태 관리
- `useEffect`: 사이드 이펙트 처리 (스크롤 이벤트)

### CSS 유틸리티
- Tailwind CSS 클래스 사용
- `cn()` 헬퍼 함수로 조건부 클래스 적용

### 외부 라이브러리
- `lucide-react`: 아이콘 컴포넌트 제공
