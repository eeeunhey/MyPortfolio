# 📌 SkillsSection 카테고리 필터링 기능 추가

## 1) 카테고리 배열 정의
| 코드 | 설명 |
|---|---|
| `const categories = ["all", "frontend", "backend", "tools"];` | 스킬 분류용 카테고리 배열 정의. `"all"`은 전체 표시, 나머지는 각 분류만 필터링 |

---

## 2) 상태(state) 관리
| 코드 | 설명 |
|---|---|
| `const [activeCategory, setActiveCategory] = useState("all");` | 현재 선택된 카테고리를 저장하는 상태. 기본값 `"all"` |

---

## 3) 필터링 로직
| 코드 | 설명 |
|---|---|
| `const filterSkills = skills.filter((skill) => activeCategory === "all" \|\| skill.category === activeCategory);` | 선택된 카테고리에 따라 `skills` 배열을 필터링. `"all"`이면 전체 출력 |

---

## 4) 카테고리 버튼 UI
| 코드 | 설명 |
|---|---|
| `{categories.map((category, key) => ( ... ))}` | 각 카테고리를 버튼으로 렌더링 |
| `onClick={() => setActiveCategory(category)}` | 버튼 클릭 시 해당 카테고리로 상태 변경 |
| `className={cn(..., activeCategory === category ? "bg-primary text-primary-foreground" : "bg-secondary/70 text-foreground hover:bg-secondary")}` | 현재 활성화된 버튼은 `bg-primary` 강조, 비활성 버튼은 흐린색 표시 |

---

👉 이로써 `SkillsSection`은 **카테고리 버튼 클릭 → 해당 기술만 표시** 되는 기능이 추가됨.
