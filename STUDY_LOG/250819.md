{meteors.map((meteor) => (
  <div
    key={meteor.id}                        // React가 리스트 렌더링할 때 중복 방지용 고유 key
    className="meteor animate-meteor"      // CSS 클래스 → 유성 스타일 + 애니메이션 효과
    style={{                               // 인라인 스타일 → meteor 객체의 속성값을 이용
      width: meteor.size + "px",           // 크기 (가로) *50 별똥별이 더 길어진다 신기!
      height: meteor.size + "px",          // 크기 (세로)
      left: meteor.x + "%",                // X좌표 (화면 가로 위치)
      top: meteor.y + "%",                 // Y좌표 (화면 세로 위치)
      animationDelay: meteor.delay,        // 애니메이션 시작 지연 시간
      animationDuration: meteor.animationDuration + "s", // 애니메이션 실행 시간
      position: "absolute",                // 화면 절대 위치 지정
    }}
  />
))}


해석

--animate-meteor → 사용자 정의 CSS 변수 이름

meteor 5s linear infinite → 애니메이션 속성 값

meteor → @keyframes meteor 라는 애니메이션 이름

5s → 애니메이션 실행 시간 (5초)

linear → 속도 곡선 (일정한 속도)

infinite → 무한 반복 실행

즉,

.meteor {
  animation: var(--animate-meteor);
}


이렇게 작성해 두면, 각 요소가 --animate-meteor 변수에 설정된 애니메이션을 따라 움직이게 돼요.

왜 CSS 변수를 쓰냐?

유연성 – 클래스마다 다른 애니메이션을 적용할 때 변수 값만 바꿔주면 됨.

재사용성 – 공통 애니메이션 정의 후 여러 곳에서 사용 가능.

동적으로 변경 가능 – JavaScript로 변수 값(--animate-meteor)을 바꿔서 속도/패턴 제어 가능.

예시
:root {
  --animate-meteor: meteor 5s linear infinite;
}

.meteor {
  animation: var(--animate-meteor);
}

@keyframes meteor {
  from {
    transform: translate(0, 0);
  }
  to {
    transform: translate(100px, 300px);
    opacity: 0;
  }
}


👉 이러면 .meteor 요소들이 5초마다 대각선으로 흘러가면서 사라지는 "별똥별 효과"가 나와요 ✨

    transform: rotate(60deg) translateX(0); 별이 돌아가버림

✅ 구현 순서 & 핵심 기능 설명

네비 항목 정의 (navItem)
네비게이션에 표시될 메뉴 이름과 이동할 위치/주소를 배열로 관리합니다.
장점: 항목 추가/수정이 쉬워지고, .map() 렌더링으로 코드가 깔끔해집니다.

유틸 import (cn)
cn은 조건부 클래스 묶어주는 유틸(예: classnames)입니다. 스크롤 여부에 따라 동적으로 클래스를 토글할 때 유용합니다.

컴포넌트 기본 골격
Navbar 함수형 컴포넌트를 만들고, 상단 고정/전이 효과 등 공통 클래스를 부여합니다.
스크롤 상태 관리 (isScrolled)

useState(false)로 초기값 설정.

스크롤이 10px 초과되면 true, 아니면 false가 되어 배경/블러/테두리 등 스타일을 바꿉니다.

스크롤 이벤트 등록 (useEffect)

handleScroll에서 window.scrollY > 10 체크.

addEventListener('scroll', ...)로 구독, return의 클린업에서 removeEventListener로 해제.

주의: 의존성 배열 []을 넣어 마운트/언마운트 시 1회 등록/해제하도록 해야 메모리 누수를 막습니다.

클래스 동적 적용 (cn)

isScrolled면 배경/블러/보더가 생기고, 아니면 투명 배경을 유지하는 식으로 시각적 전환을 줍니다.

네비 항목 렌더링 (.map)

navItem을 순회해 <a> 링크를 출력합니다.

접근성(ARIA)·키보드 포커스 스타일을 고려하면 포트폴리오 품질이 올라갑니다.

🧩 정리된 작동 코드 (React + Tailwind 예시)
// Navbar.tsx
import React, { useEffect, useState } from "react";
import { cn } from "@/lib/utils";

// 1) 네비 항목 정의
const navItem = [
  { name: "Home", href: "#hero" },
  { name: "Projects", href: "#projects" },
  { name: "Contact", href: "#contact" },
  // { name: "버튼이름", href: "주소" },
];

// 3) 컴포넌트
export const Navbar: React.FC = () => {
  // 4) 스크롤 상태
  const [isScrolled, setIsScrolled] = useState(false);

  // 5) 스크롤 이벤트 등록
  useEffect(() => {
    const handleScroll = () => {
      // ✅ screenY(X) → scrollY(O)
      setIsScrolled(window.scrollY > 10);
    };

    // 최초 진입 시 한 번 체크
    handleScroll();

    window.addEventListener("scroll", handleScroll, { passive: true });
    return () => window.removeEventListener("scroll", handleScroll);
  }, []); // 의존성 배열 필수

  // 6) 동적 클래스 적용
  return (
    <nav
      className={cn(
        "fixed inset-x-0 top-0 z-40 transition-all duration-300",
        "backdrop-blur-sm",
        isScrolled
          ? "bg-white/70 border-b border-black/10 shadow-sm"
          : "bg-transparent"
      )}
      aria-label="Primary Navigation"
    >
      <div className="mx-auto max-w-6xl px-4 h-14 flex items-center justify-between">
        {/* 로고/타이틀 */}
        <a href="#hero" className="font-semibold tracking-tight">
          My Portfolio
        </a>

        {/* 7) 네비 아이템 */}
        <ul className="flex items-center gap-6">
          {navItem.map((item) => (
            <li key={item.name}>
              <a
                href={item.href}
                className={cn(
                  "text-sm hover:opacity-80 focus:outline-none focus:ring-2 focus:ring-offset-2 focus:ring-black/30 rounded-md px-1 py-0.5",
                  isScrolled ? "text-gray-900" : "text-white drop-shadow"
                )}
              >
                {item.name}
              </a>
            </li>
          ))}
        </ul>
      </div>
    </nav>
  );
};
