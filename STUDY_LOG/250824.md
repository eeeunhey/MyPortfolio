className="text-foreground/80 hover:text-primary transition-colors duration-300"
📝 구성 요소 설명

text-foreground/80
text-foreground → Tailwind에서 --foreground라는 CSS 변수(보통 글자색) 값을 사용.
/80 → 불투명도(Opacity)를 80%로 지정. 즉, 살짝 흐린 글자색.

hover:text-primary
마우스를 올렸을 때(hover 상태) 글자색이 --primary 변수 값으로 바뀜.

보통 primary는 메인 색상(파랑, 빨강 등)으로 지정해둔 값.
transition-colors

색상(color, background-color, border-color 등) 변경이 애니메이션 되도록 설정.
그냥 바뀌는 게 아니라 부드럽게 전환됨.

duration-300
전환 애니메이션이 300ms(0.3초) 동안 실행되도록 설정.
