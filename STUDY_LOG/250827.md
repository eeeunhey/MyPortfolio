# 📌 Skills 데이터 구조

## 1) Skills 객체 스키마
| 속성 | 설명 | 예시 |
|---|---|---|
| `name` | 기술 이름 | `"React"`, `"Node.js"`, `"Docker"` |
| `level` | 숙련도 (정수값, % 의미) | `95`, `90` |
| `category` | 기술 분류 | `"frontend"`, `"backend"`, `"tools"` |

## 3) UI에 적용된 주요 Tailwind CSS 클래스
| 요소 | 클래스 | 설명 |
|---|---|---|
| Section | `py-24 px-4 relative bg-secondary/30` | 위아래 패딩 24, 좌우 패딩 4, 배경 secondary 30% 투명 |
| Container | `container mx-auto max-w-5xl` | 가운데 정렬, 최대 너비 `5xl` |
| Title | `text-3xl md:text-4xl font-bold mb-12 text-center` | 반응형 폰트, 볼드, 하단 여백, 중앙 정렬 |
| Grid | `grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-6` | 모바일 1열 → 태블릿 2열 → PC 3열, 간격 6 |
| Card | `bg-card p-6 rounded-lg shadow-xs card-hover` | 카드 배경, 패딩, 둥근 모서리, hover 효과 |
| Skill name | `font-semibold text-lg` | 굵게, 글자 크기 `lg` |
| Progress bar 배경 | `w-full bg-secondary/50 h-2 rounded-full overflow-hidden` | 너비 100%, 높이 2, 둥근 막대, 배경 보조색 |
| Progress bar 진행 | `bg-primary h-2 rounded-full origin-left animate-[grow_1.5s_ease-out]` | 진행 바 색상 primary, 왼쪽 기준 애니메이션 효과 |
