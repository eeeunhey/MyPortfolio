![Animation](https://github.com/user-attachments/assets/f790e5a8-32c9-492c-ab53-c8ef3d9a1f54)# 🌌 Portfolio Website

나만의 포트폴리오를 소개하는 웹사이트입니다.  
React + Vite + TailwindCSS를 기반으로 제작했으며,  
애니메이션과 인터랙션을 통해 유저가 즐겁게 볼 수 있도록 구현했습니다.

---

## 🚀 Features
- 소개(About Me) 섹션
- 프로젝트 리스트 & 상세 설명
- 블로그/아티클 링크
- 애니메이션 효과 (예: Meteor falling effect ✨)
- 반응형 디자인 (모바일/PC 지원)

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, TailwindCSS
- **Animation:** CSS Keyframes, Framer Motion
- **Deployment:** Vercel / Netlify
- **Version Control:** Git, GitHub
