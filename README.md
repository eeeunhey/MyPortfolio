# 🌌 Portfolio Website

나만의 포트폴리오를 소개하는 웹사이트입니다.  
React + Vite + TailwindCSS 기반으로 제작했으며,  
애니메이션과 인터랙션을 통해 사용자에게 즐거운 경험을 제공하도록 구현했습니다.  
또한 다크 모드와 라이트 모드를 모두 지원하여, 어떤 환경에서도 편안하게 볼 수 있습니다. 🌗

---

## 🚀 Features
- 소개(About Me) 섹션  
- 프로젝트 리스트 & 상세 설명  
- 블로그/아티클 링크  
- 애니메이션 효과 (예: Meteor falling effect ✨)  
- 반응형 디자인 (모바일/PC 지원)  
- **다크/라이트 모드 지원**

---

## 🛠️ Tech Stack
- **Frontend:** React, Vite, TailwindCSS  
- **Version Control:** Git, GitHub
