React + Vite + TailwindCSS 기반으로 제작한 포트폴리오 웹사이트입니다.  
단순히 보여주는 데 그치지 않고, 사용자 환경을 고려해 **라이트·다크 모드 전환 기능**을 추가했습니다.

TailwindCSS의 `dark:` 클래스를 사용해 모드별 스타일을 적용했고,  
React의 상태(state)를 이용해 버튼을 누르면 라이트 ↔ 다크 모드가 전환되도록 만들었습니다.  
이 과정을 통해 CSS 프레임워크 활용법과 React 상태 관리 방식을 직접 경험하고 이해할 수 있었습니다. 🌗

---

## 🚀 Features
- 소개(About Me) 섹션  
- 프로젝트 리스트 & 상세 설명  
- 블로그/아티클 링크  
- 애니메이션 효과 (예: Meteor falling effect ✨)  
- 반응형 디자인 (모바일/PC 지원)  
- **다크/라이트 모드 지원**
